CSS Grid Periodic Table

Codepen Link

View Github Repo

This is a representation of the Periodic Table of Elements and is entirely laid out using CSS Grid. The table is generated through an AJAX call to a Github repo containing the raw information in JSON. Using Greensock Animations I set up a fade in upon loading of the table and also a fade and scaling effect on the elements.

Multilingual Calendar/Clock with CSS Grid and Flexbox Fallback

Codepen Link

View Github Repo

This is a multilingual clock and calendar built on Javascript. It allows the user to select between English, French, and Spanish languages for the interface, to change the start day of the week, and to change the date order from Month/Day/Year to Day/Month/Year. This includes a Flexbox fallback for browsers that don't support Grid yet.
