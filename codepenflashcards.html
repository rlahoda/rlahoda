About This Project

https://codepen.io/rlahoda/full/xVaWpd/

Platform: Codepen
Key Technologies: Flexbox, Javasccript

This is a flashcard app that allows the user to enter information at the top for each cardas well as a front and a back, then the text is added to the "cards" below. The user is able to add cards as they go to have as many cards as they want.

When the user is done entering cards, they can hide the entering fields and only view the cards and then they can click on the cards to reveal the reverse side.

This was a project done to expand my understanding of Javascript and is done in vanilla Javascript.
